MediaForge Cast Bridge

El Bridge permite que un Chromecast o Google TV acceda a un archivo de tu PC mediante una URL temporal dentro de tu red local.

Uso en Windows

  1. Instala Python 3 si todavía no está instalado.
  2. Arrastra el vídeo o audio sobre START_CAST_BRIDGE.bat.
  3. Acepta el permiso del Firewall de Windows únicamente para redes privadas.
  4. La URL Cast se copiará al portapapeles.
  5. En MediaForge abre Transmitir, pega la URL y elige la pantalla.
  6. Mantén la consola abierta mientras reproduces. Pulsa Ctrl+C al terminar.

Privacidad y seguridad

El servidor expone únicamente el archivo elegido, con una ruta aleatoria. No sube nada a Internet y admite las peticiones parciales necesarias para avanzar o retroceder.

Úsalo solo en una red doméstica o privada. Cierra el Bridge al terminar. No abras el puerto en el router ni lo publiques en Internet.

Compatibilidad

El Bridge transporta el archivo, pero no cambia el códec. Para Chromecast, la combinación más fiable es MP4 con vídeo H.264 y audio AAC. Convierte previamente el archivo desde MediaForge cuando sea necesario.
