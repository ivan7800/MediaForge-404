# Código fuente de componentes GPL

Este paquete incluye `@ffmpeg/core` 0.12.10, licenciado como GPL-2.0-or-later.

La receta reproducible utilizada está fijada en `source/package-lock.json`. Tras ejecutar `npm install`, el paquete original queda disponible en `node_modules/@ffmpeg/core`, y `npm run build` copia su núcleo WebAssembly sin modificarlo funcionalmente.

Para una redistribución pública o comercial, conserva esta información, la versión exacta, el texto GPL y consulta las obligaciones de proporcionar código fuente correspondiente de FFmpeg y de las bibliotecas enlazadas. Este documento no constituye asesoramiento jurídico.
